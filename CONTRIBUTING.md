# Contributing Guidelines

Thank you for your interest in this repository! This workspace is dedicated to tracking my personal daily machine learning journey and is synced directly with my personal Kaggle account.

---

## 🔒 Proprietary Project Notice
* **Read-Only / Study Space:** Because this repository tracks a personal educational path, we do not accept direct code modifications, file uploads, or adjustments to existing Jupyter Notebooks.
* **All Rights Reserved:** Code snippets, training layers, and notebook analytics are proprietary. Please do not republish or clone these workflows for external competition submissions.

---

## 💬 How to Share Insights or Suggestions
Even though direct code changes are closed, collaboration and feedback are highly encouraged! If you spot an optimization, a statistical error, or a way to reduce loss curves in my models:

1. **Open a Discussion:** Navigate to the **Discussions** tab on this repository.
2. **Share Your Feedback:** Suggest alternative algorithms, hyperparameters, or data preprocessing frameworks (e.g., "You can handle the missing data in `titanicml.ipynb` using a KNN Imputer instead of mean filling").
3. **Collaborate:** We can brainstorm alternative feature engineering strategies together in the chat.

Thank you for respecting the tracking setup of this machine learning journey! 🤖
